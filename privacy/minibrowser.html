Reflow for Multitasking

(Previously called, Mini-Browser for Multitasking)

What do we do with the requested permissions?

  • Load the website/search you request
  • Modify the request to get the mobile version
  • Implement workarounds to expand the number of usable websites

What do we do with the user data?

  • No user data is collected or transmitted
  • User preference is stored on the device
  • You can permanently wipe it off by removing the extension or clearing browser cache
