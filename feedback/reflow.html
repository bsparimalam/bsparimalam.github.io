Sideview Plus is a solution for people with single monitors. When you cram a browser window to the side of a single screen and load a website, the page often doesn't fit in. This is despite the page probably having a responsive version for mobile devices.

This extension requests that responsive version of the web page on narrow windows. The threshold for what's "narrow" is customizable and you can defer requesting mobile web pages on certain domains/subdomains/pages. By default, the extension requests mobile web pages on windows narrower than 40% of your screen width and defers requesting mobile pages for well-known responsive webpages.
